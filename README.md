# Dropbox Ignore

For Windows only

## Usage

```
Usage: di.exe [status|ignore|reset] [folder]
Default folder: node_modules
Default action: ignore
```

Example

```bash
cd /path/to/folder

# Ignore "abc" folder
di ignore abc
```
